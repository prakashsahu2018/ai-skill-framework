# ai-skill-framework

Reusable Claude Code skills for development teams.

## Quick Start

```shell
# Add marketplace (one-time)
/plugin marketplace add prakashsahu2018/ai-skill-framework

# Install skills
/plugin install ai-skill-framework@ai-skill-framework
```

## Available Skills

| Skill          | Description                                                    |
|----------------|----------------------------------------------------------------|
| `java-backend` | Java 25 + Spring Boot 3 REST APIs, JWT security, service logic |
